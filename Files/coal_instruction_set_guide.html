Basic Computer vs Your ISA — Side by Side
FeatureBasic Computer (Morris Mano)Your Custom ISA
ArchitectureSingle accumulator (AC)Multi-register (R0–Rn)
SubtractionNo SUB — use CMA + INC + ADD (2's complement)SUB Rd, Rs (direct)
MultiplicationNo MUL — must program with shifts and addsMUL Rd, Rs (lower 16-bit)
OR operationNo OR — derive via De Morgan: NOT(AND(NOT a, NOT b))OR Rd, Rs (direct)
XOR operationNo XOR — 3-instruction workaroundXOR Rd, Rs (direct)
Dedicated compareNo CMP — manual subtraction and flag checkCMP Rd, Rs (sets Z/C/N flags)
Shift operationsCIR / CIL (circular, 1-bit per instr)SHL/SHR #n (arbitrary n)
Immediate loadNo LDI — requires memory round-tripLDI Rd, #imm (6-bit zero-ext)
Register-indirect memoryI-bit indirect (memory address in memory)LDM/STM via register pointer
Conditional branchesSPA/SNA/SZA/SZE (skip-next style)JZ / JNZ (direct jump)
Flag controlCLE onlyCLC/STC/CLZ (carry & zero control)
Register swapNot availableSWAP Rd, Rs (via TR temp)
Instruction width16-bit fixedCustom (5-bit opcode based)
I/O modelProgram-controlled (flag polling)Program-controlled + IEN interrupt
Key insight: Basic Computer is a teaching tool — minimal by design so you can trace every microoperation on paper. Your ISA extends it toward a practical RISC-style design, trading circuit simplicity for programmer productivity and instruction efficiency.
